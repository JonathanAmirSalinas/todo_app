# Todo Notes App

A Note Taking App

## Features

Write Notes
Read Notes
Edit Notes
Delete Notes
Sql/Local Storage

### Getting Started

This project allows a user to take notes. Using sql to save your notes on local storage, and have access to you rnotes on the go.
